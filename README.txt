## window1.py (streamlit에서 Page 1) 
- 이건 굳이 안해도 되는 부분인것같은데, 데이터 계속 열어보고 확인하기 헷갈려서 만들어봄. 
- Data2에 있던거는 
- Aggrid 사용해서 변수 별 sorting 기능, 검색기능 있음 (grouping도 가능) 
- 변수 별 결측치 확인 가능 
- 변수 별 data type 확인 가능 (나중에 코딩할 때 편하게) 
- 변수 별 count plots (변수 선택 가능)
